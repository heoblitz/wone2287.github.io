`0 1 2 3`

공백이 있는 여러 값을 한꺼번에 입력받으려면

python2

```python

a, b, c ,d = raw_input.split()

```  
  

python3
```python

a, b, c ,d = input.split() 

```  
<br/>

인풋값은 str 로 저장되기 때문에 별도의 캐스팅 과정이 필요하다.             
```python
>>> type(a)
<type 'str'>

```
